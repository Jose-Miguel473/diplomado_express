# Documentación del proyecto diplomado

## 1. clonar el proyecto
```
git clone https://github.com/Jose-Miguel473/diplomado_express.git
```

## 2. Instalar dependencias
```
npm install
```

## 3. Copiar variables de entorno
```
cp .env.sample .env
```

## 4. Levantar el proyecto
### 4.1 En desarrollo
```
npm run dev
```
### 4.2 Producción
```
npm run start
```
